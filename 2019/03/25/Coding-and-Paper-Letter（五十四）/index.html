42.R语言包outsider，将外部程序集成到可部署的R工作流中可能具有挑战性。 尽管有许多有用的函数和包（例如base :: system（））用于从替代语言调用代码和软件，但这些方法要求用户独立安装相关软件，并且可能无法跨平台工作。 outsider旨在通过允许用户在所有操作系统上安装，运行和控制R之外的程序来简化这一过程。

outsider
